Квизы и игры на весь класс

Quiz «Миллионер»: факты · пропуски · парафраз (15 вопросов, 50:50, несгораемые, подсказка — текст).

Stage 4: The Weakest Link — 2–8 имён, по желанию две команды (два банка), «слабое звено» — решает ведущий.

Stage 5: Tic-tac-toe + gaps (EN UI) — две команды, имена по сторонам, 9 вопросов с пропуском из четырёх текстов.

100 to 1: board + card pool (EN) — пять «топ»-ответов по каждому тексту, дистракторы, команды A/B; тот же золотой / тёмный стиль, что у квизов.

Слова и угадайки (EN)

Glass bridge (Squid, EN): 9 ступеней, два «стекла» → слово из текстов — пары из четырёх текстов; ошибка = падение и сброс.

Red / green (Squid, EN): 12 подсказок — «из текстов» vs «декой» (без процентов; только слова и коллокации).

Alias (EN UI): 23 запоминающиеся фразы (из текстов) — имена, 1:00, ведущий отмечает угаданные; пустая колода → новый раунд; таблица лидеров.

Pictionary (EN UI): те же 23 фразы — холст; ведущий кликает угаданную фразу среди отвлечений, 1:30, Pass; таблица лидеров.

Говорение и цепочки

Speaking (Squad, EN): фраза + side quest (задания в стиле вечеринки по теме юнита) — опционально таймер + бип, без логина. · имена по порядку (сначала список имён → каждому 1:00 по очереди).

Whisper / broken telephone (EN): один телефон, цепочка имён — секрет из той же колоды; каждый вводит следующий «шёпот»; в конце — полная выкладка. Мультиплеер не нужен: устройство по кругу.

RU: память и снежный ком

What is missing? (RU UI): память + «свет погас» — выбор 5 / 10 / 15 фраз в сессии, полотно, пропадает целая строка (например «fish stocks»), вводите целиком; ошибка → сессия с нуля. Таймер показа строк: 5 / 10 / 15 с (по умолчанию 10).

Снежный ком (RU, урок): имена + 1–4 круга — банк фраз, «Дальше» → кто говорит + новая фраза; «хвост» цепочки.

Стол: домино и Eco-Uno

Dominoes (EN UI, class): two halves (A–D) + sentences — как классическое домино; открытые концы, имена, pass, boneyard. Интерфейс на EN.

Eco-Uno (EN, themes A–D): фраза + спецкарты — wild, +2, skip, reverse. Правила (RU) на странице; колода в u11-eco-uno.js.
