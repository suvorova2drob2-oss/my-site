Stage 5: Tic-tac-toe (gap-fill)

Rosters can be any size (1×1, 1×2, 3×2…). Each go is one person (big name on screen), then the next go goes to a named player on the other side, rotating through that team’s list. Pick a square, then the gap. Correct = that team’s mark; wrong = the go passes to a named player on the other team.

Each side needs at least one name — a single name on one line is enough (1 vs 1, 1 vs 2, 3 vs 2… are all fine). Order in the list = who is “up” next for that team. Separate names with new lines, commas, or semicolons.

Crosses (X) — who is on this team?

Examples: one line: Maria · or: Anna / Ben / Chris · or uneven: three names vs two on the other team.

Noughts (O) — who is on this team?

Same. Even a single name here + a single name on the left = a valid 1×1 game.
